Aquário Electrónica

Fundada em 1988, a Aquário Electrónica tem duas lojas no Porto e uma em Braga.

Aquário Electrónica tem um objetivo simples: Fornecer aos clientes todo o tipo de soluções confiáveis e acessíveis não só de Electrónica como também de Consumo.

Projetos

Tecnologias

Backend: Python

Frontend: PyQT5 (biblioteca Python)

Base de dados: SQL Server

Tecnologias adicionais: REST API

Integração de Web Services

O foco principal deste projeto foi a integração de web services da E-goi com a empresa.

As tarefas englobaram o desenvolvimento de processos automatizados para campanhas de e-mail, a criação de segmentos e a segmentação do público-alvo, o envio de e-mails automáticos e a inserção e a remoção de clientes na plataforma da E-goi, bem como a realização de uma interface para os utilizadores.

Aplicação desktop de comparação

O objetivo deste projeto foi desenvolver uma interface de comparação de produtos entre dados já existentes na base de dados da empresa, e de dados de extraídos via web scraping.

As tarefas desenvolvidas no âmbito deste projeto envolveram a leitura de ficheiros CSV e ficheiros Excel (XLSX, XLS), a apresentação dos dados dos ficheiros numa tabela com a possibilidade de filtragem, a seleção de produtos da tabela para possível comparação de dados dos próprios produtos, selecionando os campos do produto a comparar, e a alteração de estado dos produtos na base de dados.
